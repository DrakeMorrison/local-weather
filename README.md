## How To Run
1. Clone this repo and cd into it
1. Install HTTP-Server and grunt-browserify via npm
1. cd into lib directory and run ``` grunt ```
1. In Terminal, type ``` hs -p 8090 ```
1. Navigate to *localhost:8090* in your browser
